# Mineria-de-datos-Tarea-practica-3-grupo-6
Proyecto de clasificación con KNN para predicción de fuga de clientes

Análisis del Problema e Hipótesis
 Problema: La pérdida de clientes (Churn) es un desafío crítico para las empresas de
telecomunicaciones. Identificar a los usuarios en riesgo de abandonar el servicio permite
ejecutar estrategias de retención proactivas.
 1ra Hipótesis: "El comportamiento de facturación mensual y la permanencia del cliente
son indicadores determinantes; por lo tanto, el algoritmo KNN podrá clasificar la
intención de fuga basándose en la similitud de perfiles históricos".
